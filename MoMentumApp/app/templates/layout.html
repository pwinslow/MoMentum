About

How Mo'Mentum Works.

  • Motivation

    Although change.org dedicates many resources to sharing user generated petitions, relatively few resources are directed towards helping users optimize the petition writing process. Mo'Mentum seeks to fill this void.

  • Metrics for Success

    Writing is optimized for not only overall petition success but also the rate at which a petition accumulates signatures.

  • Web Scraping and change.org API

    Over 40,000 unique petition id's were scraped from the change.org website using Python and Beautiful Soup, enabling data collection for each petition via the change.org api.

  • Natural Language Processing

    Natural language processing techniques, such as sentiment analysis and part-of-speech tagging, were leveraged to extract text based features from petitions using the Python textblob and NLTK libraries.

  • Random Forests and Gradient Boosting Regressors

    A random forest classification algorithm was used to predict overall probability of petition success while a gradient boosting regression algorithm was used to predict the number of days it would take to achieve the user's signature goal.
